# FAQ

## Q1: What is Lenovo Client Update module (LCU)?
LCU is a PowerShell module designed to manage Lenovo system updates, including drivers, BIOS/UEFI, and firmware.

## Q2: How do I install LCU?
Use the following command:
```powershell
Install-Module -Name 'Lenovo.Client.Update'
```

## Q3: Does LCU require administrative privileges?
Yes, installing and applying updates typically requires running PowerShell as Administrator.

## Q4: Can I automate updates?
Yes, you can use scheduled tasks or scripts to automate update checks and installations.

## Q5: Where can I report issues?
Report issues in the Lenovo [Enterprise Client Management Forum](https://forums.lenovo.com/t5/Enterprise-Management-Board/bd-p/sa01_eg).
