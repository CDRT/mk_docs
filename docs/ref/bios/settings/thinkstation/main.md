# Main

## General

These settings affect the appearance of the BIOS Setup interface on the system.

### **Language**

Specifies the language to be displayed.

Possible options:

1. **English** - Default.
2. Français
3. Русский
4. 中文

### **Setup Mode Select**

Specifies whether to use the graphical UI mode or the text based mode.

Possible options:

1. **Graphic** - Default.
2. Text

### **Setup Content**

Controls the level of settings displayed between just the basic settings versus the more advanced settings.

Possible options:

1. **Advanced** - Default.
2. Basic
