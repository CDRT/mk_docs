# Devices

### **Card Reader**

Whether card reader will be available in OS.

Possible options:

1. **Enabled** - Default.
2. Disabled

| WMI Setting name | Values | SVP / SMP Req'd | AMD/Intel |
| :--- | :--- | :--- | :--- |
| CardReader | Disabled, Enabled | Yes | Intel |
