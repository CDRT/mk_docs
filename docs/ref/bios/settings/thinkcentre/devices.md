# Devices

![Devices](https://cdrt.github.io/mk_docs/ref/bios/settings/thinkcentre/img/tc_devices.PNG)

Bluetooth
:  
    Possible options:

    1. **Enabled** – Default.
    2. Disabled – disables Bluetooth connections. Bluetooth will not be available in the OS.

    | WMI Setting name | Values | Locked by SVP |
    |:---|:---|:---|
    | Bluetooth  | Disabled, Enabled | Yes |

Card Reader
:  
    Possible options:

    1.  **Enabled** – Default.
    2.  Disabled – disables card reader.
