# Intel(R) Thunderbolt

![Intel Thunderbolt](https://cdrt.github.io/mk_docs/ref/bios/settings/thinkcentre/img/tc_intel_r_thunderbolt.PNG)

Intel Thunderbolt Technology
:  
    Possible options:

    1. **Disabled** – Default.
    2. Enabled

Security Level
:  
    Possible options:

    1. **Use Authorization** – Default.
    2. No Security
    3. Secure Connection
    4. Display Port Only

Wake From Thunderbolt Devices
:  
    Possible options:

    1. **Enabled** – Default.
    2. Disabled

SwSMI Support
:  Enable or disable SwSMI support in ASL code.

    Possible options:

    1. **Enabled** – Default.
    2. Disabled

Notify Support
:  Enable or disable Notify support in ASL code.

    Possible options:

    1. **Enabled** – Default.
    2. Disabled

Ignore Thunderbolt Device Option ROM
:  
    Possible options:

    1. **Enabled** – Default.
    2. Disabled
